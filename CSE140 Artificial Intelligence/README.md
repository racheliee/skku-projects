# CSE140 Aritifical Intelligence: Pac-Man Projects Overview

Welcome to the Pac-Man Projects! This series of assignments introduces you to foundational concepts in Artificial Intelligence through the lens of Pac-Man. Below are summaries of each project, their learning objectives, and highlights to guide you through your journey.


---

## Project 0: Unix/Python Tutorial

**Score**: 20/20

**Summary:**  
This project focuses on setting up your development environment and introducing basic tools essential for the course. You will learn to navigate a Unix-based system, work with Python, and use autograding tools. The assignments involve simple coding tasks like calculating fruit costs and selecting the cheapest fruit shop.

**Highlights:**  
- Learn Unix basics and Python fundamentals.
- Use tools like virtual environments, Git, and Tk for Pac-Man visualization.
- Submit assignments via an autograder and resolve common coding errors.

---

## Project 1: Search in Pac-Man

**Score**: 20/20

**Summary:**  
Develop search algorithms to navigate Pac-Man through mazes to reach specific goals. You'll implement depth-first search (DFS), breadth-first search (BFS), uniform cost search (UCS), and A* search. These algorithms will be applied to scenarios like finding food and corners in Pac-Man's world.

**Highlights:**  
- Implement foundational search algorithms and adapt them for Pac-Man's state space.
- Optimize paths using heuristics and graph-based search.
- Understand trade-offs between exploration strategies like DFS and BFS.

---

## Project 2: Multi-Agent Pac-Man

**Score**: 20/20

**Summary:**  
Take Pac-Man to the next level by introducing adversarial agents (ghosts). Design and implement minimax and expectimax algorithms to handle Pac-Man's decision-making under uncertainty. You'll also create evaluation functions for smarter gameplay.

**Highlights:**  
- Implement adversarial search algorithms like minimax and alpha-beta pruning.
- Explore probabilistic modeling with expectimax for random agent behavior.
- Create better evaluation functions for decision-making in multi-agent settings.

---

## Project 3: Reinforcement Learning

**Score**: 20/20

**Summary:**  
Learn how agents can improve through experience using reinforcement learning. Implement value iteration and Q-learning to train Pac-Man to make optimal decisions in various environments, including grid-based worlds and classic Pac-Man layouts.

**Highlights:**  
- Implement value iteration and Q-learning for Markov Decision Processes (MDPs).
- Experiment with exploration-exploitation trade-offs using epsilon-greedy policies.
- Generalize learning through feature-based approximations in larger state spaces.

---

## Project 4: Pac-Man Capture the Flag

**Score**: 20/20

**Summary:**  
The final project transforms Pac-Man into a team-based competitive game. Design agents that collaborate to capture food while defending their own. This multi-player project emphasizes strategy, probabilistic tracking, and efficient computation under time constraints.

**Highlights:**  
- Develop agents that balance offensive and defensive strategies in team settings.
- Handle incomplete information with probabilistic tracking.
- Participate in nightly tournaments and refine your agents based on competition results.

---

Each project builds upon the last, equipping you with skills in search, decision-making, learning, and teamwork in AI. Dive into the code, experiment with strategies, and make Pac-Man the ultimate champion! 

For detailed instructions, visit each project's directory and refer to the provided README and source files.
