# CSE113: Parallel and Concurrent Programming

This repository contains my work for the assignments in CSE113. The assignments focus on applying parallel and concurrent programming principles, exploring topics such as threading, synchronization, parallel algorithms, and GPU programming.

---

## Assignments

### **Homework 1: Loop Unrolling and Parallelization**
- **Objective**: Optimize a computational task using loop unrolling techniques and introduce parallelism with C++ threads.
- **Key Concepts**: Loop unrolling for performance enhancement, managing multiple threads, and synchronization for shared data access.
- **Output**: A benchmark comparison of the optimized versus unoptimized code.

---

### **Homework 2: Producer-Consumer with Synchronous Queues**
- **Objective**: Implement a synchronous queue (CQueueSync) to manage communication between producer and consumer threads.
- **Key Concepts**: Thread synchronization, blocking operations (enqueue and dequeue), and ensuring data consistency.
- **Output**: A working implementation of producer-consumer queues using C++.

---

### **Homework 3: Asynchronous Producer-Consumer with Circular Buffers**
- **Objective**: Design an asynchronous producer-consumer queue with a circular buffer for batch processing.
- **Key Concepts**: Asynchronous operations, circular buffer design, batch enqueue and dequeue operations, and ensuring performance efficiency.
- **Output**: An efficient queue implementation supporting batch operations.

---

### **Homework 4: GPU Programming with WebGPU**
- **Objective**: Explore GPU programming by implementing matrix multiplication and element-wise operations using WebGPU.
- **Key Concepts**: Introduction to GPU acceleration, parallel processing on GPUs, and WebGPU development environment setup.
- **Output**: A WebGPU-based solution to accelerate matrix computations.

---

### **Homework 5: Combining Parallelism and Concurrency**
- **Objective**: Implement a hybrid system that integrates parallelism and concurrency to solve a computationally intensive task.
- **Key Concepts**: Combining multi-threading and GPU acceleration, managing concurrent execution flows, and optimizing for performance.
- **Output**: A hybrid solution showcasing the power of parallelism and concurrency in modern computing.
